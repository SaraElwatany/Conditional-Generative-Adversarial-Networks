# Generative-Models

Assignment:
Given a Face image you are required to apply the following changes to it:
a. change hair style and color
b. change age of the person
c. change facial expressions
d. change pose of the head

Deliverables:
a. A detailed report documenting the implementation steps, explanations, experimental results, and conclusions.
b. Generated Images
c. Github Repository of the code used with A readme containing how to reproduce the results
